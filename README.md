## Telegram-бот кондитера

Функции:
- приём заявок (вес, размер, пожелания, контакты);
- витрина с товарами (админ добавляет/убирает позиции с фото и описанием);
- портфолио работ (фото/карточки, которые добавляет админ);
- админ-панель для управления товарами, портфолио и заявками;
- рассылка по всем пользователям + авторассылка при добавлении нового товара;
- авто-постинг новых товаров в Telegram-канал витрины.

### Установка

1. Установите Python 3.10+.
2. В папке проекта выполните:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

3. Создайте файл `.env` на основе `.env.example` и заполните:
   - `BOT_TOKEN` — токен бота из BotFather;
   - `ADMIN_IDS` — список Telegram ID администраторов через запятую (узнать ID можно у бота `@userinfobot`);
   - `CHANNEL_ID` — канал для постинга новых товаров (можно указать `@username` или numeric id вида `-100...`).

   Важно: добавьте бота администратором в этот канал, иначе он не сможет туда писать.

### Запуск

```bash
.venv\Scripts\activate
python bot.py
```

Бот запускается в режиме long polling и использует локальную SQLite-базу `bot.db` в корне проекта.
